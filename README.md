# 荒境纪元：文字生存冒险（中文网页版）

这是一个 **HTML + JavaScript** 的中文文字生存冒险小游戏，适合在网页环境直接运行。

## 玩法特色

- 开局生存：砍伐木材、收集食物。
- 多地点冒险：可在霜裂湖、青杉林地、古代遗迹、兽人营地中选择目的地。
- 四季与时间：包含春夏秋冬与清晨/正午/黄昏/深夜循环。
- 冬季机制：冬天湖面结冰、农田冻结，部分作物无法采收。
- 城镇系统：可进入城镇，与铁匠/药师/镇长等 NPC 互动。
- 货币系统：使用金币进行补给、强化与研究。
- 建筑系统：包含工坊、仓库、法师塔等建筑发展。
- 物品栏：记录草药、木材束、法力药剂、冰晶等物资。

## 运行方式

直接用浏览器打开 `index.html` 即可。

如果你希望本地起一个简单服务，也可以：

```bash
python3 -m http.server 8000
```

然后访问：`http://localhost:8000/index.html`
