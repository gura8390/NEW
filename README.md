# 荒境纪元：文字生存冒险（中文网页版）

现代化 UI 的中文网页文字生存冒险游戏。

## 本版重点

- 开局仅进行：**命名 + 天赋选择**（不再开局选职业/路线）。
- 阵营改为在游戏内通过“加入阵营”行动选择，并在大本营获得对应技能。
- 重做为 **24 小时制**，所有核心行为采用固定耗时推进。
- 城镇已作为 **外出冒险目的地**（`灰石城镇`），在城镇内进行 NPC 互动。
- 科技研发需要先建立 **科研台**，再消耗资源研发。
- 资源系统：木头、食物、铁、煤炭、硝石、金币。
- 装备栏：头盔、胸甲、裤子、鞋子、武器。

## 外部数据文件

- `NPC.json`：NPC 名称、技能描述、可执行服务（城镇与阵营大本营导师）。
- `ITEM.json`：游戏物品清单与唯一 ID（一个物品一个 ID）。

> 页面会优先读取以上文件；若在 `file://` 直接打开导致读取失败，会使用内置回退数据。

## 运行方式

```bash
python3 -m http.server 8000
```

访问：`http://localhost:8000/index.html`
