# 荒境纪元：文字生存冒险（中文网页版）

## 本次更新重点

- 新增多套可切换界面（场景）：
  - `BFYK`：大本营界面（加入阵营、学习技能、训练）。
  - `IGVF`：城镇界面（NPC 交互：铁匠/药师/镇长）。
  - `家园`：建造并互动（床铺、堆肥筒、水井、厨房）。
  - `野外地点`：霜裂湖、林地、遗迹、矿洞等。
- 前往不同地点后可切换到对应场景并进行特殊交互。
- 新增人物需求：**饥饿、口渴、休息**，并参与日终结算。
- 物品栏可交互：可使用食物/饮品，装备可穿戴。
- 工具限制与耐久：
  - 伐木需要斧头、挖矿需要镐子、种地需要锄头。
  - 武器/工具/装备含耐久，损坏后失效。
- 开局赠送：**粗木斧 + 粗木镐**。
- 科技树调整为：**木器 → 铁器 → 钢铁**。
- 冒险可发现更强武器：如**精铁剑、远古剑**。

## 数据文件

- `NPC.json`：NPC 名称、技能描述、可执行服务。
- `ITEM.json`：全部物品，按编号规则 `itm_<category>_<3-digit>` 分类并唯一编号，方便查表。

## 运行

```bash
python3 -m http.server 8000
```

访问：`http://localhost:8000/index.html`
