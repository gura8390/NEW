# 荒境纪元：文字生存冒险（中文网页版）

这是一个 **HTML + JavaScript** 的中文文字生存冒险小游戏，适合在网页环境直接运行。

## 玩法特色

- 开局生存：砍伐木材、收集食物。
- 逐步建设：建设农田、发展科技、升级庇护所。
- 外出冒险：遭遇敌人并获得奖励或承受损失。
- 世界观元素：支持人类、兽人、精灵、矮人等种族。
- 阵营系统：晨光同盟、钢铁部族、暮影议会。
- 成长路线：武力修行 / 奥术研习。

## 运行方式

直接用浏览器打开 `index.html` 即可。

如果你希望本地起一个简单服务，也可以：

```bash
python3 -m http.server 8000
```

然后访问：`http://localhost:8000/index.html`
